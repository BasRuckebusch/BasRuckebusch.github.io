Cellular Automata

Source code

A simple cave generation system that I made in Unity.

It uses cellular automata to generate cave-like structures, allowing for customization through rule adjustments.

I developed this project in my free time. It's a simple cave generation tool that utilizes cellular automata to generate cave-like structures. Cellular automata involves defining rules for individual cells, with behavior determined based on their neighbors. We apply rules that determine whether a cell lives or dies, depending on the number of neighboring cells. By tweaking these rules and running the process for a few iterations, we can create the basic shape of a cave, which we can build off of for cave generation.

If you would like more technical details, you can read the readme at the github repository. I've been able to use this project in a handful of contexts for the purposes of terrain generation.
